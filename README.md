# mnister_for_lobe
Perform MNIST using Lobe.

## ■Requirements
- [Lobe](https://lobe.ai/)
- Local PC (Browser)

## ■Starting Method
- Start Lobe and put it in Play state.
- git clone https://github.com/PoodleMaster/mnister_for_lobe
- Launch index.html directly from your browser.
